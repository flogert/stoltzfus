# Stoltzfus Tile & Design

Marketing landing page for Stoltzfus Tile & Design — a tile installation company based in Lancaster, PA.

Original Figma design: https://www.figma.com/design/lmF97rcVk19O48JXHYQQUE/Tile-Animation-Recreation

## Tech stack

- **React 18** + **TypeScript**
- **Vite** (dev server & build)
- **Motion** (`motion/react`) — page animations and animated tile gallery
- **Radix UI Accordion** — accessible FAQ / services accordion
- **Lucide React** — icon set (Phone, Mail, Menu, X)

## Getting started

```bash
npm install
npm run dev       # starts dev server on http://localhost:3000
npm run build     # production build → build/
```
